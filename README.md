# StudyFlow – Organizador Acadêmico

O **StudyFlow** é um aplicativo completo para organização acadêmica, desenvolvido para ajudar estudantes a gerenciar suas aulas, tarefas, provas e desempenho escolar de maneira intuitiva e moderna. Com uma interface adaptável (modo claro/escuro), login seguro e sincronização em nuvem, o **StudyFlow** é o seu assistente pessoal de estudos.

---

## Recursos

O **StudyFlow** oferece uma série de funcionalidades voltadas para produtividade e organização estudantil:

- **Login seguro e perfil do usuário**: Cadastro com nome, e-mail e senha via **Supabase Auth**.
- **Modo claro e escuro**: Escolha entre temas que se adaptam ao seu ambiente e preferência.
- **Sincronização em nuvem**: Seus dados sempre seguros e acessíveis em diferentes dispositivos via **Supabase**.
- **Resumo diário**: Visualize as aulas do dia, tarefas pendentes e receba uma saudação personalizada (“Bom dia, Isabeli!”).
- **Calendário inteligente**: Visualização semanal e mensal com marcações de aulas, tarefas, provas e eventos.
- **Tarefas e provas organizadas**: Crie e visualize tarefas com prioridade, prazos e status. Registre também provas, simulados e trabalhos que serão adicionados automaticamente ao calendário.
- **Desempenho acadêmico**: Acompanhe sua evolução com gráficos por matérias e alertas.
- **Chat OPENAI Router**: A interação com o sistema por meio de chat facilita o acompanhamento das atividades e desempenho.

## Tecnologias Utilizadas

O **StudyFlow** foi desenvolvido com tecnologias modernas para performance, segurança e praticidade:

- **Flutter**: Aplicativo mobile nativo para **Android** e **iOS**.
- **Supabase Auth**: Sistema de login seguro e autenticação de usuários.
- **Supabase Database**: Banco de dados em tempo real com sincronização automática.
- **Tema Dinâmico**: Suporte a tema claro/escuro com persistência.
- **Provider**: Gerenciamento de estado eficiente.
- **Drawer Menu (hambúrguer)**: Navegação fluida e intuitiva entre seções.

## Como Executar

Para rodar o **StudyFlow** em seu dispositivo, siga os passos abaixo:

### Passos para executar o aplicativo Flutter:

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/seu-usuario/studyflow.git
   cd studyflow

2. **Instale as dependências**:

   ```bash
   flutter pub get
   ```

3. **Execute o app**:

   ```bash
   flutter run
   ```
